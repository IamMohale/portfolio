Qphela: Danger Zone Alert Mobile Applicaion

Overview: Qphela is an app designed to keep users safe by notifying them about nearby danger zones. Users can also report new danger zones, helping others stay informed in real-time. I worked on the server-side development of the app as part of the ICEP WIL Program. My Role: As the backend developer, I set up the server, integrated push notifications to alert users about danger zones, and built a MySQL-driven system to store and manage data about users and zones. I also developed the APIs that connect the app to the backend. Key Features:

  • Real-time alerts when users are near a danger zone.
  • The ability for users to add and view danger zones.
  • A database-driven system for managing location data.
Context: This project was developed during the ICEP WIL Program, giving me the opportunity to apply my technical skills to a real-world safety solution.

restViz

restViz is a lightweight library designed to create a dynamic web interface for Express applications. It automatically lists all defined API routes, providing an interactive interface explore and document your API.

Contribution to restViz: Dynamic Route Management I enhanced the route management system in the restViz project. My modification to the updateRoutes function ensures that only unique API routes are added to the documentation and testing interface, preventing the duplication of routes. Key Updates:

  • Modified the updateRoutes function to check for existing routes before adding new ones.
  • Ensured that each route is unique based on its path and method.
  • Improved the overall reliability of the API documentation by avoiding redundant entries.
